# lessterfy

Find, compile and watch .less files corresponding to your browserify import
tree.

## install

With [npm](http://npmjs.org) do:

```
npm install lessterfy -g
```

## use

```bash
lessterfy my/file.js -o build/bundle.css -p extra/less/path
```
